PUS-PYTHA (Perpustakaan Sekolah Sederhana - Python Application)

Cara menjalankan:
1. Buka folder proyek PUS-PYTHA lewat Command Prompt atau Terminal:
    cd PUS-PYTHA
2. Jalankan program utama dan gunakan perintah:
    python pus-pytha.py
3. Program akan tampil di layar
    1. Login sebagai Admin
    2. Masuk sebagai User
    3. Keluar dari aplikasi
